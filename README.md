# Python Projects

## 1. To-Do List App
A simple command-line To-Do List application.
  
### Features
- Add, View, Update, Delete tasks
- Mark tasks as completed
- Auto-save using `tasks.json`

**Run**
```bash
python todo_list.py
```

## 2. Calculator App
A simple command-line calculator.

### Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Input validation
- Division by zero handling

**Run**
```bash
python calculator.py
```
